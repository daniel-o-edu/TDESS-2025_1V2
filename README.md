Este repositório é utilizado para centralizar os arquivos das seguintes unidades curriculares:

• TESI (Teste de Sistemas)

Também os documentos da Situação de Aprendizagem Integrada.

*** Passo a passo para clonar o repositório em casa ***
Para baixar os arquivos e trabalhar no seu computador, siga as instruções abaixo:

1. Instale o Git: Caso ainda não tenha, baixe e instale o Git em seu computador.
2. Abra o terminal: No Windows (CMD ou PowerShell) ou no terminal do Linux/Mac.
3. Escolha a pasta: Navegue até a pasta onde deseja salvar o projeto usando o comando `cd`.
4. Clone o repositório: Digite o comando abaixo e pressione Enter: `git clone [URL_DO_REPOSITORIO]`
5. Acesse a pasta: Entre na pasta criada com: `cd [NOME_DA_PASTA]`
6. Para atualizar a pasta no seu computador, digite `git pull`
   
Agora você já pode acessar todos os materiais das aulas que serão atualizados semanalmente!
